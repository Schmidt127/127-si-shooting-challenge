# Radio — 2025–26

Twelve market media kits for Montana radio stations.

Each folder contains:

- `RADIO-MEDIA-KIT.md` — on-air talking points, athlete list, season stats
- `EMAIL-TO-STATION.txt` — draft email to program director

Regenerate: `python tools/airtable/_build_radio_media_kits.py`

See `BUILD-SUMMARY.md` for last run metadata.
