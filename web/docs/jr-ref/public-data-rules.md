# JR Ref — public data rules

What may appear on public hoopchallenges.com pages for JR Referee Clinics.

## Allowed (with publish filter)

- Participant display name (first + last or roster name)
- School / team name
- City or region (statewide program)
- Mentor display name
- Game date/time/location (when game pages exist)
- Clinic session labels (public schedule)

## Never expose

- Parent/guardian email, phone, address
- Date of birth (full) — use age group or grade if needed
- Medical notes, waiver internals, payment status
- Fillout internal IDs, admin notes, error fields
- Unpublished / withdrawn participants

## Publish pattern

Add a checkbox or status field in Airtable (e.g. `OK for Web` or `Status = Active`) and filter all public views.

Mirror Shooting Challenge: if unsure, default to **hidden**.

## Registration

Public site may **link** to Fillout forms. Do not embed PII collection without review.
